# Getting started

Create either file:

```text
src/pages/my-page/page.tsx
src/pages/my-page/page.md
```

Both become `/my-page/`.

## Markdown features

Remark GFM supports tables, task lists, autolinks, and strikethrough.

| Format | Supported |
| --- | --- |
| React TSX | Yes |
| Markdown | Yes |
