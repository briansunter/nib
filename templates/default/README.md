# My Nib site

This site uses [Nib](https://github.com/briansunter/nib) as a framework
dependency.

```bash
npm run dev
npm run typecheck
npm run build
npm run preview
```

Edit `nib.config.ts`, replace the pages under `src/pages`, and put interactive
components under `src/islands`. The deployable static output is
`dist/client`.
